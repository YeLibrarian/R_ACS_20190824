Reproducible Data Analysis and Publishing in Chemistry with R Setup Instructions

Requirements: This workshop is hands-on, so participants are encouraged to bring in and use their own laptops. There are no pre-requisites, and we will assume no prior knowledge about the tools.

Contact: Please email yel@mit.edu for questions and information not covered here.

Setup

In this workshop, we will mainly use RStudio Cloud through a browser for hands-on activities. Please watch out for an email invitation from RStudio Cloud to join our Learning Space there. Here, as a backup solution, please install the following software on your laptop. Please make sure to install everything and try opening it to make sure it works before the start of your workshop. If you run into any problems, please feel free to email the instructor or arrive early to your workshop on the first day.

This workshop will be using the software outlined in the install instructions below. Please see the section for your operating system for those directions.

  • Windows
  • Mac
  • Linux

Windows

Please go through all the installation steps below and make sure that you not only installed them, but start them up to make sure they're working.

  1. A word processing program
    For this workshop you will need a word processing program. Many people already have Microsoft Word installed, and if you do, you're set!
    If you need a word processing program, there are a few other options, like OpenOffice and LibreOffice. Install instructions for LibreOffice, which is free and open source, are here.
    • Download the Installer
      Install LibreOffice by going to the installation page. The version for Windows should automatically be selected. Click on the button below "Main Installer" Download Version x.y.z. You will go to a page that asks about a donation, but you don't need to make one. Your download should begin automatically.
    • Install LibreOffice
      Once the installer is downloaded, double click on it and it should install.
    • To use LibreOffice, double click on the icon and it will open.

  2. R
    As a backup, we may use RStudio. RStudio is a nice interface to the programming language R. To use RStudio, you need to install both R and RStudio.
    • Download R from here
    • Run the .exe file that was just downloaded
    • Go to the RStudio Download page
    • Under Installers select RStudio 1.2.1335 - Windows 7+
    • Double click the file to install it
    • Once it's installed, open RStudio to make sure it works and you don't get any error messages.

Mac

Please go through all the installation steps below and make sure that you not only installed them, but start them up to make sure they're working.

  1. A word processing program
    For this workshop you will need a word processing program. Many people already have Microsoft Word installed, and if you do, you're set!
    If you need a word processing program, there are a few other options, like OpenOffice and LibreOffice. Install instructions for LibreOffice, which is free and open source, are here.

    • Install LibreOffice
      Once the installer is downloaded, double click on it and it should install.

  2. R
    As a backup, we may use RStudio. RStudio is a nice interface to the programming language R. To use RStudio, you need to install both R and RStudio.

    • Go to the RStudio Download page

    • Once it's installed, open RStudio to make sure it works and you don't get any error messages.

Linux

Please go through all the installation steps below and make sure that you not only installed them, but start them up to make sure they're working.

  1. A word processing program
    For this workshop you will need a word processing program. LibreOffice comes preinstalled with several Linux distributions. If you don't already have it, use your package manager to install it: (e.g., sudo apt-get install libreoffice for Ubuntu and other Debian-based distributions).
  2. R
    As a backup, we may use RStudio. RStudio is a nice interface to the programming language R. To use RStudio, you need to install both R and RStudio.
    • Follow the instructions for your distribution from CRAN. For most distributions, you can use your package manager (e.g. for Debian/Ubuntu run sudo apt-get install r-base, and for Fedora run sudo yum install R) but make sure that you have at least R 3.6 (as pre-packaged versions might be out of date).
    • To install RStudio, go to the RStudio Download page
    • Under Installers select the version for your distribution.
    • Once it's downloaded, double click the file to install it (or sudo dpkg -i rstudio-x.yy.zzz-amd64.deb at the terminal).
    • Once it's installed, open RStudio to make sure it works and you don't get any error messages.
